# VIBIN Store

A modern e-commerce store built with Next.js and Supabase.

## Features

- Custom apparel printing
- Shopping cart
- User authentication
- Admin dashboard
- Ambassador program

## Tech Stack

- **Frontend:** Next.js
- **Database:** Supabase
- **Styling:** CSS

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Visit http://localhost:3000

## Pages

- `/` - Home
- `/shop` - Shop
- `/print` - Custom Print
- `/cart` - Cart
- `/login` - Login
- `/profile` - Profile
- `/admin` - Admin
- `/ambassador` - Ambassador

## Deploy

Deploy to Vercel:

```bash
vercel
```
